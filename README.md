# 앱 공지사항 플러그인 (App Announcement Plugin)

이 프로젝트는 Flutter 마이크로커널 아키텍처 데모 앱을 위한 [1단계] 정적 정보 플러그인입니다.

## 기능
- `plugin.json` 파일을 통해 정적인 공지사항 UI를 정의합니다.
- `Container`, `Column`, `Text` 위젯과 기본 스타일링 속성을 사용하여 구성됩니다.

## 사용법
이 저장소를 GitHub에 올린 뒤, 메인 앱의 플러그인 로더에 저장소 URL을 입력하여 로드할 수 있습니다.
`https://github.com/hangyuCho/mca-plugin-announcement`
